# 感谢您安装 海畅智慧 软件
=================================== 



描述:
----------------------------------- 

您使用的为海畅智慧2.0版本更加稳定，发行日期为2016年3月10日。




### 注意事项:


	1. 为了能让手机客户端 正常连接 电脑端， 请不要阻止 软件进行联网操作（连接局域网的路由器）， 
	   否则手机端无法获取电脑端内容。

	2. 可断开广域网操作，即无需连接Internet互联网。

	3. 在手机端观看视频和图片，请关闭电脑的睡眠状态，否则电脑启动睡眠将中断与手机端的数据传输。

	4. 软件必须在window64位系统上使用。




### 开发原理：



	1. 此区别于迅雷，QQ等C语言软件开发软件，安装时写注册表并直接获取连接互联网权限. 
	   本软件为纯JAVA语言开发，安装时不写注册表，只进行简单的exe解压缩包操作，开启时360软件和诺
       顿将提示是否准许软件连接网络，以获取联网权限。


	2. 软件JAVA开发环境为64位操作系统，故不支持32位操作。
