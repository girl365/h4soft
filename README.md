必看事项:

1. 第一次开启运行需要右击安装程序，找到“以管理员身份运行”（以后再次运行可直接双击运行）。

2. 第一次开启运行界面如遇到卡住，请关闭应用程序，同1操作。

3. 软件已经过诺顿，卡巴斯基，Avast等国外杀毒软件验证，
   第一次“以管理员身份运行” 国产360杀毒拦截报警请放心开启,我们已提交360误报.（见开发原理A）。
   以后再次运行360拦截无此报警提示。

开发原理：

A. 软件为JAVA语言开发，长期以来,360对JAVA程序的误报十分严重. 

B. 数据库使用H2,纯JAVA数据库，参见http://www.h2database.com,
   使用java Service Wrapper（见开发原理D），
   需要“以管理员身份运行”（见必看事项1）。

C. 文件服务核心程序H4lj，以java Service Wrapper封装(见开发原理D）,
   需要“以管理员身份运行”（见必看事项1）.

D. 软件使用java Service Wrapper将java注册为windows服务.（360杀毒拦截报警）
   参见http://wrapper.tanukisoftware.com公司官方。
