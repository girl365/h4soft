### 感谢您安装 海畅智慧 软件 ###


================================================================================


必看事项:

1. 第一次开启运行需要右击安装程序，找到“以管理员身份运行”（以后再次运行可直接双击运行）。

2. 第一次开启运行界面如遇到卡住，请关闭应用程序，同1操作。

3. 软件已经过诺顿，卡巴斯基，Avast等国外杀毒软件验证，国产360杀毒报警请放心开启（见开发原理1）。

4. 可断开广域网操作，但是手机和电脑必须在统一局域网。

5. 在手机端观看视频和图片，请关闭电脑的睡眠状态，否则睡眠状态将中断与手机端的数据传输。



================================================================================

开发原理：

A. 软件为JAVA语言开发。

B. 数据库使用H2,纯JAVA语言数据库，参见http://www.h2database.com,
   使用java Service Wrapper（见开发原理D），
   需要“以管理员身份运行”（见必看事项1）。
